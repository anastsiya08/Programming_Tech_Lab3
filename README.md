# 📊 Система анализа социально-экономических показателей РФ
### *Лабораторная работа №3 по дисциплине «Технологии программирования»*

---

## 📝 Описание проекта
Данное приложение представляет собой комплексный инструмент для анализа и визуализации статистических данных Российской Федерации. Проект выполнен в рамках командной разработки, где каждый участник отвечал за свой тематический блок (вариант), а затем результаты были интегрированы в единое GUI-приложение.

---

## 👥 Команда и зоны ответственности

### **Вариант 8 — Анастасия (@anastsiya08)**
**Объект анализа:** Среднемесячная начисленная заработная плата мужчин и женщин.

**Реализованный функционал:**
*   **Импорт данных:** Автоматическая загрузка из CSV-файла в интерактивную таблицу `QTableWidget`.
*   **Аналитика:** Расчет темпов роста заработной платы с использованием библиотеки `Pandas`.
*   **Визуализация:** Построение линейных графиков динамики зарплат.
*   **Прогнозирование:** Математическая экстраполяция данных на заданное количество лет.

### **Вариант 15 — (@shusashka)**
**Объект анализа:** Статистика инфекционных заболеваний в РФ.

**Реализованный функционал:**
*   **Обработка данных:** Работа с официальными сведениями Росстата по различным типам инфекций.
*   **Визуализация:** Построение сравнительных графиков и гистограмм заболеваемости.
*   **Анализ:** Определение периодов максимального снижения и роста показателей.
*   **Прогноз:** Реализация механизма прогнозирования будущих трендов.

## 🛠 Технологический стек
Для реализации проекта были выбраны следующие инструменты:
*   **Язык программирования:** `Python 3.10+`.
*   **Интерфейс (GUI):** `PyQt5` — для создания многовкладочного интерфейса.
*   **Анализ данных:** `Pandas` — обработка CSV, фильтрация и расчеты.
*   **Графики:** `Matplotlib` — построение графиков, интегрированных в окна программы.
